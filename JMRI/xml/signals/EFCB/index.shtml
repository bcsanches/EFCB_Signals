JMRI: EFCB - Estrada de Ferro Central do Brasil Definition

This system contains signalling definitions corresponding to the EFCB - Estrada de Ferro Central do Brasil.

Not all of the possible masts have been defined but there is a varied selection here that should cover most requirements.

See the Aspects page for full details of the available masts and aspects.
